# <p align='center'> Installation d'Archlinux sous Virtualbox </p>


--------------------------------------------------------------------------------------------------------------------------------------------
### Création de la Machine Virtuelle
```
Type d'installation: EFI
Disque-Dur: 32 Go
Mémoire-Vive: 4 Go
```

--------------------------------------------------------------------------------------------------------------------------------------------
### Installation d'Archlinux
###### Définir le clavier en FR
```bash
loadkeys fr;
```

###### Définir le mot de passe du compte root
```
passwd root;
```
